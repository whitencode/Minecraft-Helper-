# Minecraft Helper

Minecraft Helper is a web application that helps players quickly find Minecraft farms and view basic information about them.

>  **Project Status:** In Development

This project is still under active development. New farms, guides, and features will be added over time.

---

## Features

-  Search farms
-  Multi-language support
-  Dynamic loading from JSON
-  Responsive card layout
-  Smooth animations
-  Individual farm pages

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- JSON

---

## Future Plans

- Complete farm guides
- More farm categories
- Better filtering system
- Performance improvements
- Mobile optimization
- Additional languages

## Installation

Clone the repository:

git clone https://github.com/whitencode/Minecraft-Helper-.git

Open `index.html` or run the project using Live Server.
